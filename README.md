# Bitaxe
Sito ufficiale: https://bitaxe.org/

Bitaxe Gamma monta un chip BM1370 (15 J/TH di efficienza), utilizzato su S21PRO.

S21PRO ha 3 hashboard, ognuna con 65 chip, per un totale di 195 chip.
